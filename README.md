**Profile Objective:**  
This (JD) describes the core job duties of "head boner" occupation.

**Main roles / Employee tasks:**

1) Trims meat, glands, and organs from carcass heads for use in making or processing for medicinal use or byproducts, performing any combination of following tasks.

2) Places head on table or in holding hooks and trims skin meat from jaw, jowls and skull with knife.

3) Cuts fat from lean meat drops it into containers.

4) Inserts knife into mouth cavity to cut remove tongue.

5) Trims fat, ragged edges, veins, and bruises from tongue.

6) Cuts and pulls hog snout from head, using knife snout-pulling machine.

7) Pulls jaw from skull, using jaw pulling machine, bar, or chisel.

8) Places skull on block splits it with automatic splitter or cleaver.

9) Removes brains, pituitary gland, and other parts, and places them in separate trays.

10) Removes horns from head, using shears.

11) May be known according to duty performed as Brain Picker ; Chiseler, Head ; Jawbone Breaker ; Jowl Trimmer ; Snout Puller ; Splitter, Head ; Temple-Meat Cutter ; Tongue Cutter ..
